# MiniDialog 对话框

### 介绍：功能丰富、使用简单、灵活多样的轻量级对话框组件。

#### 在线示例： [http://minidialog.applinzi.com/](http://minidialog.applinzi.com/)

#### 兼容情况：Chrome45+，Firefox40+，Edge15+，Safari11+，IE11。

#### 使用方法

##### 1. 信息提示对话框

```js
## 内容可选
Dialog.info( "标题", "内容" );
Dialog.success( "标题", "内容" );
Dialog.warn( "标题", "内容" );
Dialog.error( "标题", "内容" );
```
